# Transaction Anomaly Detection Dashboard

## 🛡️ Real-time fraud detection using Isolation Forest

### Features
- **37+ Engineered Features** - Amount spikes, velocity, geographic mismatch, counterparty patterns
- **Isolation Forest Model** - Unsupervised anomaly detection
- **Interactive Dashboard** - Real-time visualization with Plotly
- **CSV Upload** - Analyze your own transaction data
- **Risk Scoring** - High/Medium/Low risk classification

### How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
